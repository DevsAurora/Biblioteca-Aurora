# Biblioteca-Aurora
Projeto Aurora: uma aplicação web para gerenciamento de acervo, usuários e empréstimos, criada com foco em usabilidade, escalabilidade e design limpo.

## Funcionalidades para usuários comuns
- Empréstimo e devolução de livros
- Consulta ao catálogo por título, autor ou categoria

## Funcionalidades para Bibliotecários 
- Cadastro e edição de livros, autores e editoras
- Gerenciamento de usuários (criar, bloquear, redefinir senha)
- Controle de estoque e exemplares disponíveis
- Relatórios de uso (livros mais emprestados, atrasos, estatísticas)

## Tecnologias
- Frontend: HTML, CSS, Javascript
- Backend: Node.js + Express
